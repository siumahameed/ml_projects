# 🚀 Logistic Regression Project

## 🔍 Predicting Rock or Mine based on sonar data

This project applies **Logistic Regression** to predict whether an object is mine or rock under the sea analyzing the data from sonar rader.
---


## 📂 Project Overview

🔹 **Algorithm Used:** Logistic Regression
🔹 **Dataset:** `sonar data.csv`
🔹 **Libraries Used:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn


🔗 **GitHub Repository:** [Project Link]([https://github.com/27abhishek27/Logistic-Regression-Project](https://github.com/siumahameed/ml_projects))

---



📌 **Performance Metrics Used:**

- **Accuracy Score**: Overall correctness of predictions.
- **Confusion Matrix**: Evaluates true positives, true negatives, false positives, and false negatives.

---

## 🔚 Conclusion

✅ **Insights Gained:**
- Logistic Regression provides a solid baseline for detection of an object, which can be improved with advanced models.
